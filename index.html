📘 SIBOLTECH User Manual

Open Source Manual

Reference guide for dashboard usage, prediction workflow, growth graphs, calibration, and history.

DASHBOARD

The dashboard shows real-time sensor readings, actuator status, and nutrient controls.

1) Sensor Readings

  • Status: Normal, Warning, Critical
  • Level: deviation from normal conditions
  • Recommendation: suggested corrective action
  • Numerical Value: live sensor value

Automation behavior: Warning = no automation trigger, Critical = automation may activate.

Sensor Threshold Summary

  • pH: <5.5 Critical, 5.5–5.8 Warning, 5.8–6.2 Normal, 6.2–7 Warning, >7 Critical
  • DO: <5 Critical, below optimal Warning, optimal Normal
  • Temperature: >30°C Critical, 28–29°C Warning, 19–27°C Normal, 15–18°C Warning, <15°C Critical
  • Humidity: >80% Critical, 70–79% Warning, 50–70% Normal, 40–49% Warning, <30% Critical
  • TDS: <500 Critical, 500–700 Warning, 700–1400 Normal, >1400 Critical

2) Actuator Status

  • Air Pump, Exhaust Fan In/Out, Grow Lights (Aero/DWC)
  • Switches auto-sync to actual relay state
  • Override mode: when enabled, system follows automated behavior only

3) Nutrient Solution Controls

  • pH Up: dispenses 1 mL/sec
  • pH Down: dispenses 1 mL/sec
  • Leafy Green: activates for 1 minute
  • Misting Pump: activates misting system

PREDICTION

  1. Select Plant, Growing Method, Date
  2. (Optional) Enter actual measurements: Height, Length, Weight, Leaves, Branches
  3. Use Run Prediction (Sensors Only) or Save Actual + Predict
  4. Use See All Predictions for saved records

SENSOR READINGS

Line charts display sensor values over time.

GROWTH STATE

  • Metrics: Height, Weight, Length, Leaves, Branches
  • Intervals: 1 Day, 14 Days, All Records
  • Legend: Green = Aeroponics, Blue = DWC, Orange = Traditional

CALIBRATION

Enable Calibration Mode first to temporarily disable automation.

  • pH: 2-point or 3-point, capture pH 4.00 / 6.86 / optional 9.18, apply calibration
  • DO: 1-point (100%) or 2-point (100% + 0%), apply calibration
  • TDS: 1-point or 2-point, input known ppm, capture, apply calibration

HISTORY

  • Plant History: Aeroponics / DWC / Traditional
  • View modes: Daily or 15-minute
  • Tables include timestamps, sensor readings, plant readings, and actuator events
