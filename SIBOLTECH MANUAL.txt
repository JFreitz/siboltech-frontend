SYSTEM USER MANUAL
DASHBOARD
The Dashboard provides a real-time overview of the system, including sensor readings, actuator status, and nutrient solution controls.
________________


1. Sensor Readings
This section displays the current condition of all sensors.
Each parameter includes:
* Status: Normal, Warning, or Critical
* Level: Indicates deviation from normal conditions
* Recommendation: Suggested action based on readings
* Numerical Value: Actual sensor measurement
Automation Behavior:
* Warning: Actuators will not automate
* Critical: Actuators will automatically activate
________________

2. Actuator Status
This section displays and controls the system actuators.
Available actuators:
               * Air Pump
               * Exhaust Fan (In)
               * Exhaust Fan (Out)
               * Grow Lights (Aeroponics)
               * Grow Lights (DWC)
Switches automatically update based on actuator activity. For example, when an actuator is turned off by the system, the corresponding switch will also turn off.
Override Mode:
               * When enabled, manual control of switches is disabled
               * The system will follow automated behavior only
________________


3. Nutrient Solution Controls
This section allows manual control of nutrient-related components.
               * pH Up: Dispenses 1 mL per second
               * pH Down: Dispenses 1 mL per second
               * Leafy Green Solution: Activates for 1 minute
               * Misting Pump: Activates the misting system
________________


PREDICTION
This section provides plant growth predictions based on sensor data.
Step 1: Input Selection
               * Plant Number (1–6)
               * Growing Method (Deep Water Culture or Aeroponics)
               * Date
________________


Step 2: Add Actual Measurements (Optional)
Users may input actual plant data:
               * Height
               * Length
               * Weight
               * Number of Leaves
               * Number of Branches
________________


Step 3: Actions
               * Run Prediction (Sensors Only)
 Displays prediction results without actual data
               * Save Actual + Predict
 Saves input data and compares with predicted values
               * See All Predictions
 Displays all stored prediction records
________________


Output Display
After running a prediction, the following are shown:
                  * Sensor input summary
                  * Prediction versus actual comparison table
                  * Bar graph visualization

CALIBRATION
Before performing calibration, enable Calibration Mode to temporarily disable actuator automation.
________________


pH Calibration
                  1. Select either 2-point or 3-point calibration
                  2. Enter water temperature
                  3. Place the sensor in buffer solutions and capture readings:
                  * pH 4.00
                  * pH 6.86
                  * (Optional) pH 9.18
                  4. Click Apply Calibration
________________


Dissolved Oxygen Calibration
                  1. Select 1-point or 2-point calibration
                  2. Enter water temperature
Options:
                  * 1-point: Capture 100% (air saturation)
                  * 2-point: Capture 100% and 0%
Click Apply Calibration
________________


TDS Calibration
                  1. Select 1-point or 2-point calibration
                  2. Enter water temperature
                  3. Input the known ppm value of the solution
                  4. Click Capture
                  5. (Optional) Repeat for a second solution
                  6. Click Apply Calibration
________________
